## 📌 About The Project

**Premium BMI Tracker & Health Dashboard** is a modern, responsive, and visually polished web application that helps users calculate their **Body Mass Index (BMI)** and understand their health category in a simple, interactive way.

The app goes beyond a basic calculator by adding a beautiful BMI gauge, goal-based health suggestions, unit conversion, result history, light/dark theme support, and wellness tips for a balanced lifestyle. 🌱

---

## 🚀 Key Features

✅ **Instant BMI Calculation** using height and weight  
📏 **Multiple Height Units**: `cm`, `m`, `ft`, `in`  
⚖️ **Multiple Weight Units**: `kg`, `lbs`, `st`  
🎯 **Goal-Based Advice** for general health, weight loss, weight gain, and maintenance  
📊 **Animated BMI Gauge** with category indicator  
💚 **BMI Category Detection**: Underweight, Normal, Overweight, Obese, Severe Obese  
🧮 **Ideal Weight Range** based on user height  
🌙 **Dark/Light Theme Toggle** with saved preference  
💾 **Save BMI Results** using browser local storage  
📜 **History Modal** to view, delete, or clear saved records  
🔒 **Privacy Friendly**: data stays in the user’s browser  
📱 **Fully Responsive Design** for desktop, tablet, and mobile  
💡 **Wellness Tips** for hydration, activity, nutrition, sleep, and study-life balance

## 🧠 BMI Formula

The app calculates BMI using the standard formula:

```text
BMI = weight (kg) / height² (m²)
```

### 📊 BMI Categories

| BMI Range | Category |
|----------|----------|
| Below `18.5` | 🥗 Underweight |
| `18.5 - 24.9` | ✅ Normal |
| `25 - 29.9` | ⚠️ Overweight |
| `30 - 39.9` | 🏃 Obese |
| `40+` | 🩺 Severe Obese |

2. Open the project folder:

## 🌟 App Highlights

### 🎨 Premium Interface

The UI uses a smooth glassmorphism style, glowing accents, animated elements, and a clean dashboard layout to make health tracking feel simple and engaging.

### 🔁 Smart Unit Conversion

Users can switch between height and weight units, and the app converts entered values automatically for a smoother experience.

### 💾 Local History

Saved BMI records are stored in the browser through `localStorage`, allowing users to review their progress without creating an account.

### 🔐 Privacy First

No user data is sent to a server. BMI results and history remain inside the user’s own browser.
